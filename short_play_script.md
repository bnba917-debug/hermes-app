# 短剧：《后妈的秘密》

> **类型**：悬疑 / 反转  
> **时长**：约 2 分 30 秒  
> **角色**：小凯（男，28岁）/ 阿姨玲（女，35岁）  
> **场景**：客厅 / 一个场景到底

---

## 剧本正文

### 【0:00 - 0:30 开端】

**[客厅，灯光暖黄。桌上摆着两杯红酒和一碟水果。]**

**阿姨玲**（背对镜头，切水果）：你回来了？

**小凯**（换鞋，随意）：嗯，今天谈了个客户，累死了。

**阿姨玲**（转身，端着水果走到沙发前，语气平静）：小凯，我问你个事儿。

**小凯**（坐下，拿起酒杯）：说呗。

**阿姨玲**（盯着他）：你相信这个世界上……有鬼吗？

**小凯**（愣了下，笑）：你恐怖片看多了吧？

**阿姨玲**（面无表情）：你回答我就行。

**小凯**（喝酒，随口）：信啊，你就是鬼，漂亮女鬼，满意了吧？

**阿姨玲**（微微一笑）：那你猜对了。

---

### 【0:30 - 1:30 发展】

**[阿姨玲从口袋里缓缓拿出手机，点开一个视频，把屏幕转向小凯。]**

**小凯**（眯眼看）：这什么……（突然瞪大眼睛）

**[镜头切到手机屏幕特写：一个昏暗房间里，阿姨玲和一个陌生男人亲密拥抱。男人转头——竟然是阿杰，小凯已故的亲哥。]**

**[镜头切回小凯震惊的脸]**

**小凯**（声音颤抖）：这是什么时候的？你……你跟我哥……

**阿姨玲**（淡定坐下，翘起腿）：三年前吧，那时候我跟你哥在一起。

**小凯**（猛地站起来）：那你他妈为什么嫁给我？你故意的？！

**阿姨玲**（冷笑）：是啊。找上你这么好骗的货色，不容易呢。

**小凯**（一把摔碎酒杯）：你这个骗子！你到底想干什么？！

**阿姨玲**（不慌不忙）：为了你哥的东西。你们家那套老宅子，遗嘱上写的是谁？你自己清楚。

---

### 【1:30 - 2:00 高潮反转】

**小凯**（咬牙切齿）：所以你从跟我谈恋爱那天起，就在算计？

**阿姨玲**（站起身，走到他面前，压低声音）：你哥死之前，我来不及说。但你们家的东西……凭什么全归你？

**小凯**（突然冷静下来，嘴角浮起一丝笑）：你说完了？

**阿姨玲**（皱眉）：什么？

**小凯**（从口袋里掏出一支录音笔，按下播放键）

**[录音笔里，传来阿杰的声音]**

---

### 【2:00 - 2:30 终极反转】

**阿杰（录音）**：
> "凯弟，你听到这段录音的时候，我可能已经不在了。我跟你说个事——你嫂子，不对，你那个女朋友，她一直在打听咱们家的老宅。别信她，她说的每一句都是在演戏。哥对不起你，把她留给了你……"

**阿姨玲**（脸色骤变，后退一步）：你……你什么时候录的？

**小凯**（收起录音笔，冷笑）：我哥临死前给我的。他什么都告诉我了。

**阿姨玲**（声音发抖）：不可能……他说过他爱我……

**小凯**（拿起红酒，一饮而尽，放下酒杯）：爱不爱你我不知道。但我知道——你想算计我们家的东西，下辈子吧。

**[阿姨玲瘫坐在沙发上，表情从惊恐变成绝望，最后化作冷笑。]**

**阿姨玲**（低声）：好……算你们兄弟厉害……

**[特写：小凯转身离开，背影消失在门缝里。桌上那杯红酒，在灯光下像血一样红。]**

**[画面渐黑]**

**【全剧终】**

## 📋 拍摄指引

| 项目 | 建议 |
|------|------|
| 场景 | 一个封闭客厅，暖色调灯光 |
| 配乐 | 开头温馨 → 中途阴森低音 → 反转后加重鼓点 |
| 关键道具 | 手机（视频）、录音笔、酒杯 |
| 台词节奏 | 前半段松弛 → 争吵爆发 → 后半段冷静对峙 |
| 特写镜头 | 1. 手机视频画面  2. 小凯震惊表情  3. 录音笔按键  4. 最后那杯红酒 |

---

## 🧠 反转逻辑

```
第一层：阿姨玲 ↔ 死鬼阿杰有染 → 小凯是受害者
第二层：阿杰留录音笔 → 小凯一直在演戏 → 阿姨玲才是被算计的那个
```
