2. Hermes · 一次对话时序图

面试画这张图，体现你懂 流式 和 异步写记忆
sequenceDiagram
    autonumber
    participant App as App
    participant SDK as SDK
    participant GW as Gateway
    participant RT as Agent Runtime
    participant R as Redis
    participant V as 向量记忆
    participant LLM as 推理池

    App->>SDK: 发送消息 + session_id
    SDK->>GW: HTTPS / SSE
    GW->>GW: 解析 JWT，注入 user_id
    GW->>RT: 转发请求
    RT->>R: 加载会话上下文
    RT->>V: 检索用户长期记忆 filter=user_id
    RT->>LLM: 组装 Prompt 流式生成
    loop 流式 Token
        LLM-->>RT: delta
        RT-->>SDK: SSE event
        SDK-->>App: 渲染
    end
    opt 需要 Tool
        RT->>RT: 执行 Tool 沙箱 user 上下文
        RT->>LLM: 继续生成
    end
    RT-->>App: done
    RT->>R: 更新会话 异步
    RT->>V: 摘要写入长期记忆 异步

10. 银行 · HITL 中断恢复时序

sequenceDiagram
    participant U as 用户/坐席
    participant G as LangGraph
    participant M as MemorySaver
    participant T as 银行核心 Tool

    U->>G: 办理购买理财
    G->>G: 决策调用敏感 Tool
    G->>M: interrupt_before 持久化 State
    G-->>U: 展示确认页 暂停
    U->>G: 人工确认通过 thread_id
    G->>M: 恢复 State
    G->>T: 执行 Tool
    T-->>G: 结果
    G-->>U: 返回办理结果
    Note over G,M: 全链路审计日志
